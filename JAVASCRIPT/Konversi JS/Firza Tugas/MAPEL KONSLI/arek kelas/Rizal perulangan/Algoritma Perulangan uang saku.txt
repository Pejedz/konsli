Algoritma Perulangan uang saku

Nama: Rizal Dewa W. /25

1. Inisialisasi Array:

	Buat array DataUang untuk menyimpan data uang saku siswa.

2. Meminta Jumlah Data Siswa:

	a.) Tampilkan prompt kepada pengguna untuk memasukkan jumlah data siswa yang ingin diinputkan.
	b.) Simpan hasil input pengguna ke dalam variabel jumlahSiswa setelah dikonversi menjadi integer.

3. Loop untuk Memasukkan Data Siswa:

	a.) Buat loop yang berjalan sebanyak jumlahSiswa.
	b.) Di dalam loop:
		1.) Minta pengguna memasukkan nama siswa melalui prompt.
		2.) Simpan input pengguna ke dalam variabel namaSiswa.
		3.) Minta pengguna memasukkan jumlah uang saku yang dibawa oleh siswa tersebut melalui prompt.
		4.) Simpan input pengguna ke dalam variabel uangSaku.
		5.) Simpan objek yang terdiri dari namaSiswa dan uangSaku ke dalam array DataUang.

5. Menampilkan Data:

	a.) Setelah loop selesai, tampilkan semua data siswa dan uang saku yang telah diinputkan dalam bentuk tabel di konsol menggunakan console.table(DataUang).
